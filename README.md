     git clone             https://github.com/prosperOjonimi/CompanyReviewDashboard.git
    cd CompanyReviewDashboard

Open index.html in your browser (use a live server for API requests).

🚀 What I Built and Why

I built a simple dashboard with vanilla JavaScript, HTML, and CSS.

It allows users to:
	•	Search for companies using the Company Search API
	
•	Click a company to view its reviews using the Reviews API
	
•	Save favorite companies to localStorage for persistence
	
I built this because it demonstrates real-world skills like API integration, state management, and responsive UI design.
